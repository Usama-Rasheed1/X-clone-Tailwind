						X.com Homepage Clone

This repository contains a clone of the homepage of X.com (formerly known as Twitter), built using Tailwind CSS. The project demonstrates a responsive and modern layout that replicates the X.com homepage design and functionality.

Features

- Responsive design
- Modern layout
- Utilizes Tailwind CSS for styling

Technologies Used

- [Tailwind CSS](https://tailwindcss.com/)
- [Vite](https://vitejs.dev/) for development
- [Node.js](https://nodejs.org/) for the development environment

Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/x-clone-tailwind.git
   ```

2. Navigate to the project directory:

   ```bash
   cd x-clone-tailwind
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to view the homepage.
